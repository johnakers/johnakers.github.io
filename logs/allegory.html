Allegory

Allegory is a game I've wanted to make for a little bit. Its an RPG, without the battles. As this project is a work in progress, consider the below a README, and TODO list, for now. I'm building this in JavaScript, leveraging Drew Conley's Tutorial as there are certain parts of canvas, that I don't quite know. Additionally, the aim is to use game development patterns, rather than more web development ones, though I know there will be some crossover, at times.

The game is inspired by JRPGs like Final Fantasy Tactics (a favorite), and the story is loosely based on Vampire Hunter D Bloodlust.

More will be put on this page, as the game develops.
